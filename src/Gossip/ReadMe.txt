													***********ReadMe*****************





	To Compile type javac *.java -cp apache-log4j-1.2.17/log4j-1.2.17.jar

	Note: Log4j folder should be placed in the src folder only along with the log4j.properties file.

	To run type :java -cp .:apache-log4j-1.2.17/log4j-1.2.17.jar application

	Note : Some System.out.println comments have been intentionally left so
	that we could use them during testing purposes.
